利用hexo搭建自己的博客网站

搭建自己博客网站方法：

1、准备工作：
① 首先保证自己的电脑中有Node环境（下载Node即可）；
② 我用的git版本控制工具进行的指令操作（下载git即可）；
③ $ cnpm install hexo-cli -g ； 在全局安装hexo-cli；
④ $ hexo init blog 创建一个blog文件夹；（此处速度可能较慢）

2、搭建自己的本地blog仓库；
按照顺序输入指令：
$ cd blog
//当前文件夹下；
$ cnpm install;
//创建依赖；
$ hexo server
//可在本地启动服务预览

3、配置远端服务器Git：
① $ cnpm install hexo-deployer-git –save
//安装hexo-deployer-git 工具；

4、上传自己的博客吧：

$ hexo new post one.md
在blog\source_posts\one-md.md文件中进行编辑即可；

执行指令：传入远端；
$ hexo g
$ hexo d

自己的博客轻松上传至网站中。
